# Ionic Sidemenu App
This is a sample Ionic 2 App to demonstrate left and right Sidemenues.


# Installation
```bash
$ git clone https://github.com/ionic-toolbox/app_from_scratch_with_sidemenu_left_and_right.git
$ npm install
$ ionic serve
```
